空間: O(n)

climbStairsV2

時間: O(n)
空間: O(1)

基本上就是費氏數列，原因如下：

Base Cases:
n == 1時只有一種走法 1
n == 2時只有兩種走法 1+1 / 2

n == 3時，走法為f(1) + f(2) = 3
n == k時，走法為f(k-2) + f(k-1)

原因：
解 n == k時，可以想成從k-1出發，或從k-2出發兩種可能
若從k-1出發，則只有唯一解就是再踏一步
若從k-2出發，則有兩個解，1+1或2，但如果選1+1的話，就會和k-1的case重複，要扣掉，這樣一來k-2也只剩下唯一解
所以f(k)就等於f(k-1) + f(k-2)
