maxProfit

時間: O(n)
空間: O(1)

解釋：
cur += prices[i] - prices[i - 1]：不斷累加，想像成是在i-1買，然後一直累加上去，只要獲利是正的，他就不會被重置，所以不用擔心錯過最佳買點；若被重置了，就代表走到目前為止所有買點都會使獲利為負，那也沒有必要保留
res = max(res, cur)：把最佳解存起來
cur = max(cur, 0)：若獲利變成負的了，那就重置，可以想像成「不要在i-1買了」，直接跳到下一個可能獲利的買點

e.g.
prices = 7 5 4 3 1 2
  i    = 0 1 2 3 4 5
 cur   = X 0 0 0 0 1
 res   = 0 0 0 0 0 1


prices = 7 5 3 4 8 9
  i    = 0 1 2 3 4 5
 cur   = X 0 0 1 5 6
 res   = 0 0 0 1 5 6
